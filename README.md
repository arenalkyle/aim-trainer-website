# Aim Trainer Website
A performance-oriented 2D browser-based training application designed to improve mouse accuracy and reaction time. Built with a focus on low-latency input handling and scalable difficulty.

* **Language:** TypeScript  
* **Frontend:** HTML5 Canvas / CSS3 
* **Storage:** JSON-based persistent state

## Features (In Development)
* **Customizable Target Behavior:** Adjustable scaling and movement patterns for varied training scenarios.
* **Real-Time Rendering:** Optimized rendering logic using a TypeScript-based architecture to ensure high frame rates.
* **Persistent Tracking:** A leaderboard and profile system to track session statistics and user settings.
